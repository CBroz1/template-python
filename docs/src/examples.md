# Examples

This page documents the example modules included in the template. When you fork
and rename the template, replace these with modules relevant to your project.

## Fibonacci

`template_python.examples.fibonacci` provides utilities for computing Fibonacci
numbers.

```python
from template_python.examples.fibonacci import f, f_list

f(10)       # returns the 10th Fibonacci number: 55
f_list(10)  # returns the first 10 Fibonacci numbers: [1, 1, 2, 3, 5, ...]
```

See also `notebooks/example.ipynb` for an interactive walkthrough.
